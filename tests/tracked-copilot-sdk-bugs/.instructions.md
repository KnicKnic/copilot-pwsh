# Tracked Copilot SDK Bugs

Each bug repro file must be **self-contained** — it should compile and demonstrate the bug with no dependencies outside this project (except the `GitHub.Copilot.SDK` NuGet package).

## Rules
- Every bug is a separate `.cs` file implementing `IBugRepro` (defined in `Program.cs`).
- Each file must be fully self-contained — no shared helper files. Duplicate code across repros is fine.
- Do not reference CopilotShell or any other project — this is a standalone console app meant to be shared with the SDK team.
- Include a header comment block documenting: the bug, expected vs actual behavior, and the `dotnet run` command to execute it.
- Do **not** run these tests automatically — only run when the user explicitly asks.
